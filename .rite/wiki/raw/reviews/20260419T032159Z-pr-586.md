- 指示書 (commands/wiki/init.md) の Edit ツール instruction が hardcode する anchor コメント `# <<< gitignore-wiki-section-end (anchor / F-09 対応)` は、本プラグイン自己開発 repo の `.gitignore` L131 のみに存在。consumer project には `templates/` 経由の distribution 経路も、`/rite:init` や `gitignore-health-check.sh` による inject 経路もない。
